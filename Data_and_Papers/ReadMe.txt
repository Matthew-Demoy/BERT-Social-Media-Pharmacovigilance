This Folder Conatins all current Datasets acquired to train and test our models(ADE Classifier, Sentiment ADE)
Each sub folder should contain:

A link to the Paper or Origin of the dataset(for documentation and thesis writing purposes)
The raw data in the form of user ids and tweet ids
The data in the form of Columns with the tweet text pulled from twitter
The data in the form of a csv,tsv file to be used in the model.
The data with missing tweets pulled from the data set.

An additional folder to hold lexicons/vocabularies will be in this folder as well.
